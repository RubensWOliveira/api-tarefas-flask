# API de Tarefas com Flask

Projeto de API REST desenvolvido com Flask e SQLAlchemy para gerenciamento de tarefas.

## Funcionalidades

* Criar tarefa
* Listar tarefas
* Atualizar tarefa
* Excluir tarefa

## Tecnologias utilizadas

* Python
* Flask
* SQLAlchemy
* SQLite
* Postman

## Como executar o projeto

Clone o repositório:

git clone https://github.com/RubensWOliveira/api-tarefas-flask.git

Entre na pasta do projeto:

cd api-tarefas-flask

Crie o ambiente virtual:

python -m venv venv

Ative o ambiente virtual:

Windows:

venv\Scripts\activate

Instale as dependências:

pip install -r requirements.txt

Execute o servidor:

python app.py

A API estará disponível em:

http://localhost:5000
